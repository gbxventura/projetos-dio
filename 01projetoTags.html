Como funciona a tag p?

A tag p em HTML é usada para criar um parágrafo de texto. Ela é uma das tags mais comumente usadas para estruturar e formatar o conteúdo textual em uma página da web.

Ao utilizar a tag p, você está indicando ao navegador que deseja que o texto contido dentro dessa tag seja tratado como um parágrafo. O navegador, então, renderizará o texto e aplicará estilos padrão, como um espaçamento antes e depois do parágrafo, para melhorar a legibilidade.

Lembrando que a tag p é um elemento de bloco, o que significa que, por padrão, ela ocupa uma linha inteira em um layout. Se você deseja exibir vários parágrafos em uma única linha, pode utilizar CSS para modificar seu comportamento, como alterando a propriedade display para inline ou inline-block.

Como funciona a tag a?

A tag a em HTML é usada para criar um hyperlink, ou seja, um link que permite aos usuários navegar para outra página da web, arquivo ou âncora dentro da mesma página.

O atributo obrigatório href especifica o destino do link. Pode ser uma URL completa, um caminho relativo para um arquivo ou o identificador de uma âncora dentro do mesmo documento.

Além do atributo href, a tag a também suporta outros atributos opcionais, como target para especificar onde abrir o link (por exemplo, em uma nova janela ou guia) e title para fornecer informações adicionais quando o usuário passar o mouse sobre o link.

A tag h1 em HTML é usada para criar um cabeçalho de nível 1, que é o mais alto na hierarquia de cabeçalhos. Ela é frequentemente utilizada para títulos principais ou importantes em uma página da web.

O texto dentro da tag h1 será exibido como um cabeçalho em destaque, geralmente com uma fonte maior e mais enfatizada do que o conteúdo normal da página.

É importante notar que a tag h1 deve ser usada apenas uma vez por página para indicar o principal título ou cabeçalho da página. Ela é seguida por outros níveis de cabeçalho, como h2, h3, h4, etc., para estruturar o conteúdo de forma hierárquica.

A tag h1 também é útil para a otimização de mecanismos de pesquisa (SEO), pois os mecanismos de pesquisa geralmente consideram o texto dentro da tag como um indicador importante do tema da página.
